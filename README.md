# itblurs

AI-powered Chrome extension that detects and blurs inappropriate content in real-time.

## Features

-  Real-time video content detection
-  Local AI processing (NSFWJS)
-  Region-only or full-screen blur modes
-  Zero data sent to servers
-  Fully private and local

## installation
-open chrome
-open extensions
-search for itblurs
-download it

## Usage

1. Click the extension icon to open settings
2. Choose blur type (Region Only or Full Screen)
3. Adjust sensitivity slider (default 70%)
4. Extension automatically detects and blurs content on any webpage

## Privacy

All processing happens locally in your browser. Zero data is ever sent to any server.
